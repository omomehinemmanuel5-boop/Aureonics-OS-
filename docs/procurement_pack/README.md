# Lex Aureon Enterprise Procurement Pack

This folder contains a complete procurement pack for Fortune 500 buyer review.

## Deliverables
- `Lex_Aureon_Security_Whitepaper.md`
- `Lex_Aureon_Security_Whitepaper.pdf`
- `Lex_Aureon_Data_Processing_Agreement.md`
- `Lex_Aureon_Security_Questionnaire.md` (64 Q&A entries)
- `Lex_Aureon_Architecture_One_Pager.md`
- `Lex_Aureon_Architecture_One_Pager.pdf`
- `scripts/procurement/generate_procurement_pdfs.py` (PDF export utility)

## Rebuild PDFs
```bash
python scripts/procurement/generate_procurement_pdfs.py
```
